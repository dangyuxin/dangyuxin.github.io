Markdown 语法指南

1. 标题

使用 # 符号表示标题，数量代表标题级别，最多支持六级标题。

2. 段落和换行

段落之间用空行分隔，如果需要强制换行，可以在行尾添加两个空格。

3. 列表

无序列表

使用 -、+ 或 * 表示无序列表项。

有序列表

使用数字后跟英文句点表示有序列表项。

4. 链接和图片

链接

使用 [文本](链接) 格式来创建链接。

图片

使用 ![alt 文本](图片链接) 格式来插入图片。

5. 强调和引用

使用 *斜体* 或 _斜体_ 表示斜体，使用 **粗体** 或 __粗体__ 表示粗体。

引用

使用 > 表示引用。

6. 代码块

使用 ` 单引号包裹短代码，使用三个反引号创建多行代码块。

单行代码

7. 水平线

使用三个以上的连字符 (---)、星号 (***) 或下划线 (___) 来创建水平线。

8. 表格

使用竖线和减号创建表格。

9. 转义字符

使用反斜杠 (\ ) 来转义特殊字符。
